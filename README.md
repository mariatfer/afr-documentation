# Starlight Starter Kit: Basics
This documentation is built with [Astro](https://astro.build) using the Starlight theme for a clean, searchable docs experience.

## 🚀 Project Structure

```
src/content/docs/
├── README.mdx                  # You are here
├── 01-project-overview.mdx     # Architecture, tech stack, features
├── 02-code-fragments.mdx       # Notable code patterns and implementations
├── 03-seo-performance.mdx      # SEO system, performance strategy
└── 04-metrics.mdx              # Lighthouse metrics, screenshots
```

## 🧞 Commands

All commands are run from the root of the project, from a terminal:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Installs dependencies                            |
| `npm run dev`             | Starts local dev server at `localhost:4321`      |
| `npm run build`           | Build your production site to `./dist/`          |
| `npm run preview`         | Preview your build locally, before deploying     |
| `npm run astro ...`       | Run CLI commands like `astro add`, `astro check` |
| `npm run astro -- --help` | Get help using the Astro CLI                     |
