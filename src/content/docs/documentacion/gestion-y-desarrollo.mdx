<Aside type="note">El desarrollo se estructuró por fases, permitiendo avanzar de forma progresiva en los distintos módulos del sistema.</Aside>

- **Investigación y definición del modelo de negocio**: Finales de febrero - principios de marzo 2025.  
- **Diseño visual e interfaz web (HTML/CSS)**: Segunda mitad de marzo.  
- **Desarrollo de funcionalidades (login, carrito, personalización, archivos)**: Marzo - Abril.  
- **Base de datos y lógica backend (PHP + MySQL)**: Mitad de marzo - Principios de mayo.  
- **Pruebas y validación**: Mayo.  
- **Documentación y memoria**: Mayo - Junio.

## Gestión de recursos

<Aside type="tip">El proyecto fue desarrollado por una sola persona, abordando todas las áreas: diseño, desarrollo, pruebas y documentación.</Aside>

**Herramientas utilizadas:**

- Visual Studio Code `editor de código`
- XAMPP `entorno local: Apache, PHP, MySQL`
- phpMyAdmin `gestión de base de datos`
- Figma `diseño` 
- JavaScript puro y Splide.js `funcionalidades dinámicas`

---

## Valoración económica

<Aside type="caution">Se priorizó el uso de herramientas gratuitas para reducir costes en la fase inicial del proyecto.</Aside>

**Costes estimados (fase actual):**

- Dominio: ~15 €/año  
- Hosting básico: ~15 €/año  
- Herramientas de diseño: versión gratuita  
- Librerías JS y entorno de desarrollo: gratuitos  

**Costes previstos si se escala comercialmente:**

- Dominio personalizado + hosting profesional con certificado SSL: 150-250 €/año.
- Integración de pasarela de pago: según proveedor. (Stripe - 1,5 % + 0,25 € por transacción)
- Publicidad: 500€

<Aside type="note">Este planteamiento se basa en el desarrollo de un MVP (Producto Mínimo Viable), con escalabilidad prevista.</Aside>
