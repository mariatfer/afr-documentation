## About This Project

**Stack:** Nuxt 4 · Vue 3 · TypeScript · SCSS · Headless CMS · Vercel

**Purpose:** Public‑facing SSR website with a browsable product catalog, dynamic content pages, contact/quote forms with file uploads, full SEO instrumentation, and cookie‑consent compliance.

**Key patterns:** Generic service layer for API calls, singleton hover interaction pattern, GSAP‑based scroll reveal components, type‑safe locale loading, polymorphic button component, and contrast‑aware UI utilities.
