## Performance Wins Summary

| Optimisation | Impact |
|---|---|
| SSR rendering | First paint includes meaningful content |
| `@nuxt/image` | Automatic WebP + responsive `srcset` |
| Lazy loading | Below‑the‑fold images load only when needed |
| Singleton hover listener | One global listener instead of many |
| Self‑hosted fonts | Zero DNS lookups, no layout shift |
| Preconnect hints | Faster initial network negotiation |
| SCSS mixins tree‑shaken | Only used mixins included |
| Passive event listeners | Smooth scrolling |
| Inline critical CSS | Faster first render |
| Brotli/gzip compression | Smaller payloads |
